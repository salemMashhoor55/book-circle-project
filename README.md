# 📚 Book Circle Application

A modern, responsive social platform for book lovers built with **React.js**. This project features a seamless experience for discovering and managing book collections.

## ✨ Key Features
- **Dynamic Navigation:** Clean Navbar with integrated icons using `lucide-react`.
- **Role-Based Access:** Implemented protected routes for Admin and User dashboards.
- **Modern UI:** Responsive design optimized for all screen sizes.
- **Stability:** Fixed reference errors and improved icon integrity.

## 🛠️ Tech Stack
- **Library:** React.js
- **Icons:** Lucide-React
- **Routing:** React Router DOM
- **Styling:** CSS / Tailwind

## 🚀 How to Run
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm start` to launch the app.
